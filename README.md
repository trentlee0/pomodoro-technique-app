## pomodoro-technique-app
> 番茄时钟， 一个小Demo应用。使用 [Electron](https://www.electronjs.org/) 构建，Electron [官方文档](https://www.electronjs.org/docs)。

#### 构建
1. 安装依赖 `npm install`
2. 打包，使用的打包工具为 [Electron Packager](https://github.com/electron/electron-packager) ，打包命令 `npm run package`。
   输出的目录为当前目录的 `out` 文件夹下。
